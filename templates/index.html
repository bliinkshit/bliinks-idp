Technical information

bliinks itself is a network of connected services. the page you're viewing right now is known as "the bliinks hub". bliinks hub provides bliinks users with a central place to manage their account. connected services use a custom-spec OAuth2 API to get user data served here for their own purposes.

bliinks hub is built with Rust and uses SQLite as its database. its run on an upgraded Dell PowerEdge T410 housed within one of our admins' personal datacenters. If you'd like to find out more about the bliinks network and how it works, check out the README on our forum.
